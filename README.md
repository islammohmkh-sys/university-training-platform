# University Training Platform - Course Library Udemy v9

إضافة شاشة محتوى الكورسات مع:
- رابط Udemy
- أقسام وفيديوهات بوقت لكل فيديو
- تعيين مجموعة متدربين على كورس
- عرض المتدربين المرتبطين بالكورس

## إعدادات Render

Build Command:
```bash
npm install --legacy-peer-deps
```

Start Command:
```bash
npm start
```

## أوامر GitHub من VS Code

```bash
git init

git branch -M main
git remote remove origin
git remote add origin https://github.com/islammohmkh-sys/university-training-platform.git
```
